# Claude Skills

A collection of custom skills for [Claude Code](https://claude.ai/claude-code).

## Skills

### `scientific-writing`

Comprehensive scientific and technical writing for biotechnology R&D in industrial settings. Covers every document type in the biotech research lifecycle:

- **Abstracts** — Grant (DARPA, DOE, ARPA-E), conference, and journal abstracts
- **Government Proposals** — DARPA, DOE, BARDA-style proposals with solicitation-aligned structure
- **Commercial Proposals** — Partnership and business development proposals
- **Technical Reports** — Detailed experimental reports with methods, results, and discussion
- **Literature Reviews** — Thematic synthesis with state-of-the-art comparison tables
- **White Papers** — Thought leadership and position papers
- **Statements of Work** — Task-level scope definition with deliverables and milestones
- **Progress Reports** — Milestone-tracked quarterly/periodic reports
- **Patent Disclosures** — Invention disclosures and patent application support
- **Presentations** — Slide decks, conference talks, posters
- **One-Pagers** — Executive summaries and internal decision memos

### Core Principles

- **Data over claims** — every assertion backed by specific numbers and comparisons
- **Audience-calibrated framing** — same result, different framing for DARPA PMs vs. commercial partners vs. internal teams
- **Precision of language** — correct terminology, units, nomenclature
- **Strategic narrative** — context → gap → approach → evidence → implication
- **Honest limitations** — credibility through acknowledging what you don't know

## Installation

Add this repo as a skill source in your Claude Code configuration, or copy the `scientific-writing/` directory into your project.
